Bachelorarbeit Portal – Werkzeugerkennung und Rückgabeprüfung

Autor: Firas Meftahi

Startanleitung:

1. Python installieren
2. Projektordner öffnen
3. Virtuelle Umgebung erstellen:

   python -m venv venv

4. Umgebung aktivieren:

   venv\Scripts\activate

5. Pakete installieren:

   pip install -r requirements.txt

6. Portal starten:

   python app.py

7. Im Browser öffnen:

   http://127.0.0.1:5000

Hinweise:
- Die Modelle befinden sich im Ordner models.
- Die Overlay- und Slotanalyse-Dateien befinden sich im Ordner slot_reference.
- Für Kamera-Zugriff lokal den Browser verwenden.
- Für die Demonstration wird empfohlen, das Portal direkt auf dem vorbereiteten Laptop zu starten.
